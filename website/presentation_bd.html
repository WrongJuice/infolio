Une base de données, (ou DataBase en anglais) permet d'enregistrer des données de façon organisée et hiérarchisée. Ces données, ou informations, sont consultables, modifiables et capables d'être mise à jour à tout moment. Elles sont triés dans des tables à travers la database afin de fonctionner ensemble et d'être plus facilement traitées. Ces base de données sont très présentes dans les entreprises, voir indispensables.

Un SGBD permet d'inscrire, modifier ou supprimer les informations de la base de données. Il comporte des mécanismes pour assurer la cohérence des informations, éviter les pannes et permettre son utilisation par d'autres logiciels. Le SQL est utilisé comme langage pour travailler sur cette base de données et ainsi rechercher, ajouter, modifier ou supprimer des données dans les bases de données relationnelles.
Un exemple ! Pour stocker ses clients, produits, commandes, livreurs, livraisons, transactions et autres données, tout site de e-commerce a besoin d'une base de données.

Pour travailler dans ce domaine, un bac S ou STI2D est fortement conseillé. Après un BAC+2 en informatique (BTS SIO ou DUT Informatique), il existe plusieurs licences professionnelles permettant d'accéder à ce métier :
Gestion et traitement informatique de données massives
Administrateur de réseaux et de base de données
Gestion et traitement statistique de base de données
Base de données
Les titulaires d'un BAC+5 (Master MIAGE ou diplôme d'une école d'ingénieur) sont cependant les plus recherchés. Une expérience de quelques années de développement est recommandée.

Anecdote

Le terme database (base de données) est apparu en 1964 pour désigner une collection d'informations partagées par différents utilisateurs d'un système d'informations militaire.
